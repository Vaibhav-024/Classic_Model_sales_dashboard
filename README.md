# 📊 Sales Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing sales data to extract meaningful insights such as total sales, profit margins, top-selling products, and customer distribution. The dashboard helps businesses make data-driven decisions.

---

## 🛠 Tools & Technologies Used

* SQL (Data Extraction & Queries)
* Microsoft Excel (Data Cleaning & Preprocessing)
* Power BI (Data Visualization & Dashboard)

---

## 📊 Key Insights

* Total Sales reached **$3.25M**
* Overall Profit Margin is **39.88%**
* Total Profit generated is **$1.30M**
* Top Selling Product: **1992 Ferrari 360 Spider red**
* Highest Sales observed in **Classic Cars category (42.13%)**
* Strong customer distribution across multiple cities globally

---

## 📈 Features

* Interactive filters (Year, City, Employee, Product Line)
* Sales trend analysis by year
* Product-wise sales distribution
* Top & Bottom performing products
* Geographical customer insights using map visualization

---

## 🎯 Business Use Case

This dashboard can be used by:

* Sales Managers to track performance
* Business Analysts to identify trends
* Companies to optimize product strategy

---

## 🚀 Conclusion

The project demonstrates how raw data can be transformed into actionable insights using data analytics tools.
